Distance Meter using Infrared Light

This is a low precision distance meter that works on light reflection method. An infrared light emitting diode emits the desired wavelength light. When the light hits any object and returns to the sensor the sensor changes its output value. The op-amp then compares the analog voltage and indicates the approximate distance.

Starting Time: April 08, 2023

Duration: 3 days

Status: Completed
